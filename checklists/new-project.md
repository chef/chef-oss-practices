* Repository configuration

  * Style guide

  * LICENSE
  * TEAMS.md
  * .github/CODEOWNERS
  * .github/ISSUE_TEMPLATE.md
  * .github/PULL_REQUEST_TEMPLATE.md
* Other tasks
  * Slack integration (`/github` command)
